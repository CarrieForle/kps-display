The format textbox allows you to format KPS display with the text of your input.

This serves as a reference rather than a tutorial. If it is confusing refer to the doc online: https://example.com

%[flags][width]<placeholder>

Placeholders:
    %s is custom KPS
    %d is decimal KPS
    %x is heximal, lower case KPS
    %X is heximal, upper case KPS
    %% is literal `%`

Flags:
    `-` flag for left aligning KPS. (By default KPS is right-aligned)
    `0` flag for zero-padding
    `c` flag for custom padding

Width is a positive integer for the minimum width (number of characters) of KPS
